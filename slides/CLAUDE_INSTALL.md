# Instalação Claude Code na Máquina

Para seguir o setup oficial acessar: `https://code.claude.com/docs/en/quickstart`

# No Terminal

- Linux, MacOS `curl -fsSL https://claude.ai/install.sh | bash`
- Windows `curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd`
- MacOS(Homebrew) `brew install --cask claude-code`
