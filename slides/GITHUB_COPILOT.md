# GitHub Copilot

- Precisa de uma conta do Github e um plano no : https://github.com/features/copilot
- Instalar Extensao: GitHub Copilot Chat (Atualmente o VSCode Ja vem com um chat integrado)

## Vantagens de usar o GitHub Copilot

- Aumenta a produtividade ao sugerir código em tempo real enquanto você digita.
- Reduz tarefas repetitivas com geração de funções, testes e trechos boilerplate.
- Ajuda no aprendizado ao sugerir abordagens e padrões de implementação.
- Acelera a criação de documentação e comentários para código existente.
- Melhora o fluxo de trabalho com assistência contextual dentro do VS Code.
- Facilita prototipação rápida de ideias, comandos e scripts.
